Buttons

The buttons structure is using the html 5 button tag. We wanted the buttons to stand out from the rest of the content on the page so for the css we used the teal color as a contrast to the other colors. There is a graidient style to the button to make it look a little nicer which uses different values and shade of the original color. spacificaly #a0beba and #528881. Another lighter graidient is used when the user hovers over the button to help indicate that they can interact with this element. #c0d3d1 and #a0beba.


HTML

<button class="button span_1_of_6">
   This is a button!
</button>

CSS

.button {
background: rgb(160,190,186);
background-image: linear-gradient(to bottom, #a0beba, #528881);
border-radius: 12px;
box-shadow: 2px 2px 4px rgb(45,75,71);
font-family: 'Quicksand', sans-serif;
font-size: 15pt;
font-weight: 300;
font-style: italic;
padding: 10px 20px 10px 20px;
text-decoration: none;
border: 0;
color: white;
}
.button:hover {
background: rgb(192,211,209);
background-image: linear-gradient(to bottom, #c0d3d1, #a0beba);
text-decoration: none;
color: black;
}
.button:focus {
outline:0;
}

List

The lists elements are used for mainly for the navigation for this style guide and future web site. In the final product the design team desided there would be no use of a literal list is the use of the site. That being said the both of the navigations used in this style guide make use of the HTML list elements. If you would like to see how they are used, please look at Navigation Menu to see how to use!

Table

To keep things easy to look at there is not much styling to the table. The main thing that is styled in the table is the bottom border of the rows for separation and the aligning of the text. Otherwise the table is quite simple. When each row of the table is hovered over, the row is highlighted a lighter color value of the teal that is used throughout the site. This is to help the user focus on a specific row and for aesthetic look.

Navigation Menu

The navigation for this style guide makes use of the unordered list elements, and its CSS styles. The color of our navigation and table of contents is #91725d. The text color is white to contrast against the brown background color and when the user hovers over the list our teal highlighting color is used #528881. It uses the Quicksand font, with the items floating to the left. Other than that this is just a simple list with simple styles.

  • Home
  • Products
  • Cold Beverages

HTML

<div id="breadcrumbs">
    <ul class="crumbsList">
        <li><a href="#">Home</a></li>
        <li><a href="#">Products</a></li>
        <li><a href="#">Cold Beverages</a></li>
        <li>Iced Mocha</li>
    </ul>
</div>

CSS

#breadcrumbs ul.crumbsList {
    padding: 14px 14px;
font-size: 12pt;
    list-style: none;
    
}
#breadcrumbs ul.crumbsList li {
    display: inline;
}
#breadcrumbs ul.crumbsList li+li:before {
    padding: 10px;
    color: #b2868d;
font-weight: bold;
    content: ">";
}
#breadcrumbs ul.crumbsList li a {
    color: #885259;
    text-decoration: none;
}
#breadcrumbs ul.crumbsList li a:hover {
    color: #528881;
    text-decoration: underline;
    font-weight: bold;
}

The breadcrumb navigation should be used so that users are able to visualize how they got to a specific place on the website. It provides the function to backtrack to previously visited webpage(s). Each label is linked(except the current webpage) and on hover the label is highlighted by the color #528881 and is also underlined.

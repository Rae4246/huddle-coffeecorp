Buttons

The buttons structure is using the html 5 button tag. We wanted the buttons to stand out from the rest of the content on the page so for the css we used the green color as a contrast to the other colors. There is a graidient style to the button to make it look a little nicer which uses different values and shade of the original color. spacificaly #a0beba and #528881. Another lighter graidient is used when the user hovers over the button to help indicate that they can interact with this element. #c0d3d1 and #a0beba.

<button class="button span_1_of_6">    This is a button! </button>

Table

To keep things easy to look at there is not much styling to the table. The main thing that is styled in the table is the bottom border of the rows for separation and the aligning of the text. Otherwise the table is quite simple. When each row of the table is hovered over, the row is highlighted a lighter color value of the green that is used throughout the site. This is to help the user focus on a specific row and for aesthetic look.
